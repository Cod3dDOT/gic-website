## Getting started

Install pnpm

```bash
npm install -g pnpm
```

Clone the project

```bash
  git clone https://github.com/Cod3dDOT/gic-website
```

Go to the project directory

```bash
  cd gic-website
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
